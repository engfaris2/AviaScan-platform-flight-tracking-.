#  AviaScan: Global Flight Tracking and Data Platform

AviaScan is a robust, web-based platform designed for real-time flight tracking and comprehensive aviation data exploration. We provide users with a clear, immediate view of the world's airspace, merging instant geospatial tracking with easy access to a detailed aviation database.

This project offers a powerful tool for travelers checking flight statuses, aviation enthusiasts, and developers needing structured air traffic data.

##  Features & Functionality

### Real-Time Tracking and Visualization

* **Map:** Visualize global air traffic. Track the live position of any specific commercial flight on an interactive map.
* **Flights:** Search for and track specific flights, view statuses, departure/arrival details, and routes.

### Data Exploration Hub

Users can view and filter comprehensive database tables containing essential aviation entities:

* **Airlines:** Full list of operating carriers and their associated flights.
* **Airports:** Global hub locations, detailed airport pages, and associated flight traffic.
* **Airplanes:** Details on aircraft types, models, and usage statistics.

### User Engagement & Analysis

* **Statistics:** View aggregate data and key performance indicators related to flight traffic, airlines, and airports.
* **Account:** Personalized account management and profile settings.
* **Reviews:** A platform for user feedback and reviews.

##  Project Structure

The repository is organized to ensure clarity and separation of concerns:

| Folder | Content | Description |
| :--- | :--- | :--- |
| **`main/`** | PHP files, Application Logic | Contains the core application logic and user interface, including PHP scripts that run the website's functionalities. |
| **`dataset/`** | Initial Data Files (e.g., CSV, SQL) | Includes the static or initial datasets used to populate the database tables (flights, airlines, airports, etc.) upon setup. |
| **`example-images/`** | Screenshots | Stores all image assets and visual references used within the `README.md` file to showcase the platform's features. |
| **`README.md`** | Main Documentation File | Provides an overview of the project, features, setup instructions, and the visual showcase of the application pages. |
